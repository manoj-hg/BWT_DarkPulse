# Dataset Folder

This folder contains student face images for the face recognition system.

## Instructions:

1. Add student images in JPG, JPEG, or PNG format
2. Name files as: `StudentName.jpg` or `StudentName.png`
3. Examples:
   - `JohnDoe.jpg`
   - `JaneSmith.png`
   - `AliceJohnson.jpg`

## Requirements:

- Use clear, front-facing photos
- One face per image
- Good lighting conditions
- High resolution images work better
- Avoid sunglasses, masks, or face coverings

## Sample Images:

You can add sample images here to test the system. The system will automatically load all valid face images when started.

After adding images, click the "Reload Faces" button in the web dashboard to load the new faces.
